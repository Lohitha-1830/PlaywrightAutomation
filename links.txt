Required commands:
1.TO run all the test case
 npx playwright test
2. To run specific file
 npx playwright test ./tests/alerts.spec.js
3.To run in headed mode
 npx playwright test ./tests/alerts.spec.js --headed
4. Run on single browser
 npx playwright test --porject=chromium --headed
5. To open report
 npx playwright show-report
6. Run with tag
 npx playwright test --grep "@smoke"
7. Exclude particular tag and run everything else
 npx playwright test --grep-invert "@smoke"
8. Run tests having either @smoke or @auth:
 npx playwright test --grep "@smoke|@auth"
9. Run tests with @regression but not @slow:
 npx playwright test --grep "@regression" --grep-invert "@slow"
